==========
Networking
==========

The networking part of ``atfoundation`` is a bit slim currently,
but socket support and a HTTP client are planned.

.. note::

     The big problem we're hitting here is that we need external libs
     for the implementation of HTTP and SSL and we didn't want to add
     these as a depedency yet.
