is this an absolute or relative path

delimiter between path components

Initialize with a string, the string is split into components by the rules of the current platform

Parameters:
  • string – The string to parse
init(components: [String], absolute: Bool = false)¶

Initialize with path components

Parameters:
  • components – Array of path component strings
  • absolute – Boolean, defines if the path is absolute
init(absolute: Bool = false)¶

Initialize empty path

Parameters:
  • absolute – If set to true the empty path is equal to the root path, else it equals the current path

func join(_: Path) → Path¶

Create a new path instance by joining two paths

Parameters:
  • path – other path to append to this instance. If the other path is absolute the result is the other path without this instance.
func relativeTo(path: Path) → Path?¶

Create a path instance that defines a relative path to another path

Parameters:
  • path – the path to calculate a relative path to
Returns:

new path instance that is a relative path to path. If this instance is not absolute the result will be nil
